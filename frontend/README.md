# Lapwise

F1 race data and telemetry visualization.

## Local Development

```bash
npm install
cp .env.example .env.local
# Add your backend API URL to .env.local
npm run dev
```

## Tech Stack

Next.js 14, TypeScript, Tailwind CSS, Recharts

## Environment Variables

Required environment variables:

- `NEXT_PUBLIC_API_URL` - Backend API URL (e.g., `https://your-backend.railway.app`)

See `.env.example` for template.
